#Automatic Youtube MP3 Downloader


#Description
- This is a tool to download youtube videos and playlists automatically.
- To use, put this .py file in the same directory as a file with a list of the videos/playlists you wanted to download, separated by new lines for each entry.

#Prerequisites
- pytube
- python3

#Notes
- This code is only tested on Linux.
